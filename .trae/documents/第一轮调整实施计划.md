## 第一轮调整任务

### 1. 创建配置示例文件
- 创建 `/config/config.example.json`，包含所有配置项的示例值
- 移除敏感信息（如API密钥），使用占位符

### 2. 保护实际配置文件
- 将 `/config/config.json` 添加到 `.gitignore`
- 避免将包含敏感信息的配置文件提交到版本控制

### 3. 提取AI Prompt模板
- 创建 `/config/ai-prompt-template.txt` 存储AI提示词模板
- 修改 `ai-mapping.service.ts` 从文件读取模板
- 方便管理员查看和修改提示词

### 4. 调整映射名称显示
- 修改 `MappingFormFields.tsx` 组件
- 将"映射名称"字段改为单独占一行（使用 `col-span-2` 或 `col-span-full`）

### 5. 修复AI API调用错误
- 错误原因：火山引擎API不支持 `messages` 字段，应使用 `prompt` 字段
- 修改 `ai-mapping.service.ts` 中的 `callOpenAI` 方法
- 调整请求体格式以适配火山引擎API：
  ```typescript
  // 修改前（OpenAI格式）
  { model, messages, temperature }
  
  // 修改后（火山引擎格式）
  { model, prompt, temperature }
  ```
- 同时调整响应解析逻辑以适配火山引擎的响应格式
